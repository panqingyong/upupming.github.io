设计思想

逻辑模型 [Logistic regression]

考虑只有两个预测器 x1,x2x_1, x_2x1​,x2​ 的模型，它们可能是连续变量，也可能是布尔值。发生比对数（Logit(log-odds)）通常形式为：

l=β0+β1x1+β2x2(1)l = \beta_0 + \beta_1 x_1 + \beta_2 x_2 \tag{1} l=β0​+β1​x1​+β2​x2​(1)

系数 βi\beta_iβi​ 是模型的参数，注意这里是线性模型，因为 Logit 是预测器 x1,x2x_1, x_2x1​,x2​ 的线性组合，再加上常数 β0\beta_0β0​。相应的发生比（odds）就是指数形式：

o=bβ0+β1x1+β2x2(2)o = b^{\beta_0 + \beta_1x_1 + \beta_2x_2} \tag{2} o=bβ0​+β1​x1​+β2​x2​(2)

发生比一般定义为某个事件发生和不发生的比率 p1−p\frac{p}{1-p}1−pp​，bbb 是对数和指数函数所用的底数。因为发生比不是预测器的线性组合，这就是一个非线性的模型。

假设事件有 111 次不发生，那么就有 ooo 次发生，总共有 1+o1 + o1+o 次试验，因此事件发生的概率 ppp为：

p=o1+o=bβ0+β1x1+β2x2bβ0+β1x1+β2x2+1=11+b−(β0+β1x1+β2x2)(3)\begin{aligned} p &= \frac{o}{1 + o} \\ &= \frac{b^{\beta_0 + \beta_1x_1 + \beta_2x_2}}{b^{\beta_0 + \beta_1x_1 + \beta_2x_2} + 1} \\ &= \frac{1}{1 + b^{-(\beta_0 + \beta_1x_1 + \beta_2x_2)}} \end{aligned} \tag{3} p​=1+oo​=bβ0​+β1​x1​+β2​x2​+1bβ0​+β1​x1​+β2​x2​​=1+b−(β0​+β1​x1​+β2​x2​)1​​(3)

通常将底数 bbb 取值为 eee，但在计算指数的时候还是取 101010，这样计算出来的发生比更易于理解。对于参数为 βT=(−3,1,2)\mathrm{\mathbf{\beta}}^T = (-3, 1, 2)βT=(−3,1,2) 的模型，lll 为：

l=−3+1⋅x1+2⋅x2(4)l = -3 + 1 \cdot x_1 + 2 \cdot x_2 \tag{4} l=−3+1⋅x1​+2⋅x2​(4)

这个模型的含义如下：

  • β0=−3\beta_0 = -3β0​=−3 是 y−y-y−截距，表示所有预测器都是 0 时的发生比对数。发生比为 10−3=1:100010^-3 = 1:100010−3=1:1000，那么发生概率为 1/(1000+1)=1/1001≈0.001=0.1%1/(1000 + 1) = 1/1001 \approx 0.001 = 0.1\%1/(1000+1)=1/1001≈0.001=0.1%。

  • −β0=3-\beta_0 = 3−β0​=3 是 x−x-x−截距，表示发生比对数 l=0l = 0l=0 时 l′=1⋅x1+2⋅x2l' = 1 \cdot x_1 + 2 \cdot x_2l′=1⋅x1​+2⋅x2​ 的值。在这个时候发生比为 100=110^0 = 1100=1，发生概率为 1/(1+1)=1/2=50%1/(1+1) = 1/2 = 50\%1/(1+1)=1/2=50%。

  • β1=1\beta_1 = 1β1​=1 意味着 x1x_1x1​ 每增大 111，发生比对数增加 1⋅11 \cdot 11⋅1，发生比变为原来的 101=1010^1 = 10101=10 倍。

  • β2=2\beta_2 = 2β2​=2 意味着 x2x_2x2​ 每增大 111，发生比对数增加 2⋅12 \cdot 12⋅1，发生比变为原来的 102=10010^2 = 100102=100 倍。

这个模型很有用处，比如为了比较两种医疗方式的效果，一种将 x1x_1x1​ 减小了 333，另一种将 x2x_2x2​ 减小了 222，在这个模型之下，第一种医疗方式将发生比除以了 10(1⋅3)=100010^{(1 \cdot 3)} = 100010(1⋅3)=1000，第二种将发生比除以了 10(2⋅2)=1000010^{(2 \cdot 2)} = 1000010(2⋅2)=10000，因此第二种医疗方式更有效。

逻辑回归的任务就是估计这种逻辑模型的参数，同时计算对数据拟合的好坏程度。

假设学习步长为 η\etaη，第 kkk 轮迭代的参数为 θ(t){\mathbf{\theta}}^{(t)}θ(t)，则梯度上升迭代过程为：

θ(t+1)=θ(t)+η∑l=1mxl{yl−11+exp⁡−θ(t)Txl}(20)\theta^{(t+1)} = \theta^{(t)} + \eta \sum_{l=1}^m \mathrm{\mathbf{x^l}}\{y^l - \frac{1}{1 + \exp{-{\mathbf{\theta}^{(t)}}^T\mathrm{\mathbf{x^l}}}}\} \tag{20} θ(t+1)=θ(t)+ηl=1∑m​xl{yl−1+exp−θ(t)Txl1​}(20)

最大后验估计（MAP），带惩罚项的梯度上升法

根据贝叶斯公式：

p(ω∣D)=p(D∣ω)p(ω)p(D)(23)p(\mathrm{\mathbf{\omega}}|\mathrm{\mathbf{D}}) = \frac{p(\mathrm{\mathbf{D}}|\mathrm{\mathbf{\omega}})p(\mathrm{\mathbf{\omega}})}{p(\mathrm{\mathbf{D}})} \tag{23} p(ω∣D)=p(D)p(D∣ω)p(ω)​(23)

其中：

  • p(ω∣D)p(\mathrm{\mathbf{\omega}}|\mathrm{\mathbf{D}})p(ω∣D) 为后验概率，可以理解为『在已经知道数据的情况下模型参数为 ω\mathrm{\mathbf{\omega}}ω 的概率』
  • p(D∣ω)p(\mathrm{\mathbf{D}}|\mathrm{\mathbf{\omega}})p(D∣ω) 为似然函数，可以理解为『在给定参数情况下，得到的结果与已知数据吻合的概率』
  • p(ω)p(\mathrm{\mathbf{\omega}})p(ω) 为先验概率，可以理解为『模型参数是 ω\mathrm{\mathbf{\omega}}ω 的概率』

就拿抛硬币问题（抛了 5 次）来举例：

  • p(ω∣D)p(\mathrm{\mathbf{\omega}}|\mathrm{\mathbf{D}})p(ω∣D) 为后验概率，可以理解为『在已经知道数据的情况下（4 次正面，1 次反面）模型参数为 ω\mathrm{\mathbf{\omega}}ω 的概率』
  • p(D∣ω)p(\mathrm{\mathbf{D}}|\mathrm{\mathbf{\omega}})p(D∣ω) 为似然函数，可以理解为『在给定参数情况下，得到的结果与已知数据吻合（4 次正面，1 次反面）的概率』
  • p(ω)p(\mathrm{\mathbf{\omega}})p(ω) 为先验概率，可以理解为『模型参数是 ω\mathrm{\mathbf{\omega}}ω 的概率』

在抛硬币问题中，我们仅仅最大化似然函数是不准确的，因为求得了一个模型参数 ω\mathrm{\mathbf{\omega}}ω 使 4 次正面，1 次反面这种情况最大化了，但要知道这个模型参数出现的概率也是很小的（概率最大的模型参数是正反面 1:1 出现）。因此我们要最大化的其实是『在已经知道数据的情况下（4 次正面，1 次反面）模型参数为 ω\mathrm{\mathbf{\omega}}ω 的概率』，这就是后验概率，从 (23)(23)(23) 式中可以看出它正比于似然函数与先验概率的乘积。

在 (17)(17)(17) 式的基础上，我们可以得到后验概率：

P(θ;y∣x)∝P(y∣x;θ)P(θ)(24)P(\mathrm{\mathbf{\theta}}; \mathrm{\mathbf{y}}|\mathrm{\mathbf{x}}) \propto P(\mathrm{\mathbf{y}}|\mathrm{\mathbf{x}; \mathrm{\mathbf{\theta}}})P(\mathrm{\mathbf{\theta}}) \tag{24} P(θ;y∣x)∝P(y∣x;θ)P(θ)(24)

进一步设先验服从高斯分布：

P(θ)=sexp⁡−α2θTθ(25)P(\mathrm{\mathbf{\theta}}) = s\exp{-\frac{\alpha}{2}\mathrm{\mathbf{\theta}}^T\mathrm{\mathbf{\theta}}} \tag{25} P(θ)=sexp−2α​θTθ(25)

结合式 (17)(24)(17) (24)(17)(24)并求对数，省略常数项得到：

M(θ)~=ln⁡P(θ)+∑l=1m{ylθTxl−θTxl−ln⁡(1+exp⁡−θTxl)}=−α2θTθ+∑l=1m{ylθTxl−θTxl−ln⁡(1+exp⁡−θTxl)}=−λ2θTθ+∑l=1m{ylθTxl−θTxl−ln⁡(1+exp⁡−θTxl)}(26)\begin{aligned} \widetilde{M(\mathrm{\mathbf{\theta}})} &= \ln P(\mathrm{\mathbf{\theta}}) + \sum_{l=1}^m \{y^l{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} -{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} - \ln (1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}})\} \\ &= -\frac{\alpha}{2}\mathrm{\mathbf{\theta}}^T\mathrm{\mathbf{\theta}} + \sum_{l=1}^m \{y^l{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} -{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} - \ln (1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}})\} \\ &= -\frac{\lambda}{2}\mathrm{\mathbf{\theta}}^T\mathrm{\mathbf{\theta}} + \sum_{l=1}^m \{y^l{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} -{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}} - \ln (1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}})\} \end{aligned} \tag{26} M(θ)​​=lnP(θ)+l=1∑m​{ylθTxl−θTxl−ln(1+exp−θTxl)}=−2α​θTθ+l=1∑m​{ylθTxl−θTxl−ln(1+exp−θTxl)}=−2λ​θTθ+l=1∑m​{ylθTxl−θTxl−ln(1+exp−θTxl)}​(26)

式中取 λ=α\lambda = \alphaλ=α。

参考式 (19)(19)(19)，求导得到：

∂M(θ)~∂θ=−λθT+∑l=1mxlT{yl−P(yl=1∣xl;θ)}(27)\begin{aligned} \frac{\partial \widetilde{M(\mathrm{\mathbf{\theta}})}}{\partial \mathrm{\mathbf{\theta}}} &= -\lambda\mathrm{\mathbf{\theta}}^T + \sum_{l=1}^m \mathrm{\mathbf{x^l}}^T\{y^l - P(y^l = 1|\mathrm{\mathbf{x^l}};\mathrm{\mathbf{\theta}})\}\\ \end{aligned} \tag{27} ∂θ∂M(θ)​​​=−λθT+l=1∑m​xlT{yl−P(yl=1∣xl;θ)}​(27)

从而,带惩罚项的迭代过程为：

θ(t+1)=θ(t)+η{−λθ(t)+∑l=1mxl{yl−11+exp⁡−θ(t)Txl}}(28)\mathrm{\mathbf{\theta}}^{(t+1)} = \mathrm{\mathbf{\theta}}^{(t)} + \eta\{-\lambda\mathrm{\mathbf{\theta}}^{(t)} + \sum_{l=1}^m \mathrm{\mathbf{x^l}}\{y^l - \frac{1}{1 + \exp{-{\mathbf{\theta}^{(t)}}^T\mathrm{\mathbf{x^l}}}}\}\} \tag{28} θ(t+1)=θ(t)+η{−λθ(t)+l=1∑m​xl{yl−1+exp−θ(t)Txl1​}}(28)

牛顿法

根据多元函数求极值问题的规则，极值点处的导数一定均为 000，我们要求的是对数似然函数的极值，可以转换为求其导数的零点。在已知对数似然函数表达式（式 (19)(19)(19)）的情况下，我们可以列出 n+1n+1n+1 个方程，联立解出所有参数：

∂l(θ)∂θ0=∑l=1mx0l{yl−11+exp⁡−θTxl}∂l(θ)∂θ1=∑l=1mx1l{yl−11+exp⁡−θTxl}...∂l(θ)∂θn=∑l=1mxnl{yl−11+exp⁡−θTxl}(29)\begin{aligned} \frac{\partial l(\mathrm{\mathbf{\theta}})}{\partial \theta_0} &= \sum_{l=1}^m {x_0^l}\{y^l - \frac{1}{1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}}}\} \\ \frac{\partial l(\mathrm{\mathbf{\theta}})}{\partial \theta_1} &= \sum_{l=1}^m {x_1^l}\{y^l - \frac{1}{1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}}}\} \\ & \quad \quad ... \\ \frac{\partial l(\mathrm{\mathbf{\theta}})}{\partial \theta_n} &= \sum_{l=1}^m {x_n^l}\{y^l - \frac{1}{1 + \exp{-{\mathbf{\theta}}^T\mathrm{\mathbf{x^l}}}}\} \\ \end{aligned} \tag{29} ∂θ0​∂l(θ)​∂θ1​∂l(θ)​∂θn​∂l(θ)​​=l=1∑m​x0l​{yl−1+exp−θTxl1​}=l=1∑m​x1l​{yl−1+exp−θTxl1​}...=l=1∑m​xnl​{yl−1+exp−θTxl1​}​(29)

在具体解方程之前需要用 Hessian 矩阵来判断极值的存在性。求 Hessian 矩阵就得先求二阶偏导：

由前文的定义：

x=(x1,x2...,xm)=(x01x02x03⋯x0mx11x12x13⋯x1m⋮⋮⋮⋱⋮xn1xn2xn3⋯xnm)(n+1)×m(31)\begin{aligned} \mathrm{\mathbf{x}} &= (\mathrm{\mathbf{x^1}, \mathrm{\mathbf{x^2}}..., \mathrm{\mathbf{x^m}}}) \\ &= \begin{pmatrix} x^1_0 & x^2_0 & x^3_0 & \cdots & x^m_0 \\ x^1_1 & x^2_1 & x^3_1 & \cdots & x^m_1 \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ x^1_n & x^2_n & x^3_n & \cdots & x^m_n \\ \end{pmatrix}_{(n+1) \times m} \end{aligned} \tag{31} x​=(x1,x2...,xm)=⎝⎜⎜⎜⎛​x01​x11​⋮xn1​​x02​x12​⋮xn2​​x03​x13​⋮xn3​​⋯⋯⋱⋯​x0m​x1m​⋮xnm​​⎠⎟⎟⎟⎞​(n+1)×m​​(31)

其 Hessian 矩阵为：

H(n+1)×(n+1)=xAxT(32)\mathrm{\mathbf{H}}_{(n+1)\times(n+1)} = \mathrm{\mathbf{xAx}}^T \tag{32} H(n+1)×(n+1)​=xAxT(32)

满足 Hkj=∂2l(θ)∂θk∂θj\mathrm{\mathbf{H}}_{kj} = \frac{\partial^2 l(\mathrm{\mathbf{\theta}})}{\partial \theta_k \partial\theta_j}Hkj​=∂θk​∂θj​∂2l(θ)​，其中：

A=(hθ(x1)[hθ(x1)−1]0⋯00hθ(x2)[hθ(x2)−1]⋯0⋮⋮⋱⋮00⋯hθ(xm)[hθ(xm)−1])m×m(33)\mathrm{\mathbf{A}} = \begin{pmatrix} h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^1}}}})[h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^1}}}}) - 1] & 0 & \cdots & 0 \\ 0 & h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^2}}}})[h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^2}}}}) - 1] & \cdots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \cdots & h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^m}}}})[h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^m}}}}) - 1] \end{pmatrix}_{m \times m} \tag{33} A=⎝⎜⎜⎜⎛​hθ​(x1)[hθ​(x1)−1]0⋮0​0hθ​(x2)[hθ​(x2)−1]⋮0​⋯⋯⋱⋯​00⋮hθ​(xm)[hθ​(xm)−1]​⎠⎟⎟⎟⎞​m×m​(33)

如果 A\mathrm{\mathbf{A}}A 负定，则对任意的 n+1n+1n+1 维向量 z\mathrm{\mathbf{z}}z 有：

zTxAxTz=(xTz)TA(xTz)<0(34)\begin{aligned} \mathrm{\mathbf{z}}^T\mathrm{\mathbf{x}}A\mathrm{\mathbf{x}}^T\mathrm{\mathbf{z}} &= (\mathrm{\mathbf{x}}^T\mathrm{\mathbf{z}})^TA(\mathrm{\mathbf{x}}^T\mathrm{\mathbf{z}}) < 0 \end{aligned} \tag{34} zTxAxTz​=(xTz)TA(xTz)<0​(34)

也就是说 Hessian 矩阵 xAxT\mathrm{\mathbf{x}}A\mathrm{\mathbf{x}}^TxAxT 也是负定的。

我们可以先从二维的函数来理解牛顿迭代法,其思想是取图像上的 xxx 坐标为当前估计零点值的点为切点做一条切线，求切线与 xxx 轴的交点，将其作为新的估计零点，示意图如下：

为了求方程 f′(x)f'(x)f′(x) 的根，把原函数 f(x)f(x)f(x) 泰勒展开到二阶形式：

f(x+Δx)=f(x)+f′(x)Δx+12f′′(x)Δx2(35)f(x+\Delta x) = f(x) + f'(x)\Delta x + \frac{1}{2}f''(x)\Delta x^2 \tag{35} f(x+Δx)=f(x)+f′(x)Δx+21​f′′(x)Δx2(35)

当且仅当 Δx\Delta xΔx 无限趋于 000 时等式成立（即后面的项被省去）。此时上式等价于：

f′(x)+12f′′(x)Δx=0(36)f'(x) + \frac{1}{2}f''(x)\Delta x = 0 \tag{36} f′(x)+21​f′′(x)Δx=0(36)

因为 Δx→0\Delta x\rightarrow 0Δx→0，12\frac{1}{2}21​ 将不再起作用，于是有：

f′(x)+f′′(x)Δx=0(37)f'(x) + f''(x)\Delta x = 0 \tag{37} f′(x)+f′′(x)Δx=0(37)

求解得到：

Δx=f′(x)f′′(x)(38)\Delta x = \frac{f'(x)}{f''(x)} \tag{38} Δx=f′′(x)f′(x)​(38)

得出迭代公式：

xt+1=xt−f′(x)f′′(x),t = 0, 1, ...(39)x_{t+1} = x_t - \frac{f'(x)}{f''(x)}, \quad \text{t = 0, 1, ...} \tag{39} xt+1​=xt​−f′′(x)f′(x)​,t = 0, 1, ...(39)

在我们这里，不难得出迭代公式为：

θ(t+1)=θ(t)−UH=θ(t)−H−1U(40)\mathrm{\mathbf{\theta}}^{(t+1)} = \mathrm{\mathbf{\theta}}^{(t)} - \frac{\mathrm{\mathbf{U}}}{\mathrm{\mathbf{H}}} = \mathrm{\mathbf{\theta}}^{(t)} - {\mathrm{\mathbf{H}}}^{-1}\mathrm{\mathbf{U}} \tag{40} θ(t+1)=θ(t)−HU​=θ(t)−H−1U(40)

其中 U\mathrm{\mathbf{U}}U 为一阶导数：

U(n+1)×1=x(n+1)×m(y1−hθ(x1)y2−hθ(x2)⋮ym−hθ(xm))m×1=x1[y1−hθ(x1)]+x2[y2−hθ(x2)]+⋯+xm[ym−hθ(xm)](42)\begin{aligned} \mathrm{\mathbf{U}}_{(n+1)\times1} &= \mathbf{\mathrm{x}}_{(n+1)\times m}\begin{pmatrix} y^1 - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^1}}}}) \\ y^2 - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^2}}}}) \\ \vdots \\ y^m - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^m}}}}) \\ \end{pmatrix}_{m \times 1} \\ &= \mathrm{\mathbf{x^1}}[y^1 - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^1}}}})] + \mathrm{\mathbf{x^2}}[y^2 - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^2}}}})] + \cdots + \mathrm{\mathbf{x^m}}[y^m - h_\theta(\mathrm{\mathbf{\mathrm{\mathbf{x^m}}}})] \end{aligned} \tag{42} U(n+1)×1​​=x(n+1)×m​⎝⎜⎜⎜⎛​y1−hθ​(x1)y2−hθ​(x2)⋮ym−hθ​(xm)​⎠⎟⎟⎟⎞​m×1​=x1[y1−hθ​(x1)]+x2[y2−hθ​(x2)]+⋯+xm[ym−hθ​(xm)]​(42)

结论

  1. 逻辑回归使用牛顿法比梯度下降快很多。
  2. 为了得到线性分类边界，我们必须使数据服从高斯分布，但是经过实际测量，当不满足朴素贝叶斯假设，得到的结果人仍然较好。
  3. 黑塞矩阵的计算复杂度比较高，但是牛顿法下降的速率快，因此还是更加优。
  4. 逻辑回归能够很好地处理高维数据。
